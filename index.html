Implementation Details

This clock uses cutting-edge techniques for optimal performance and flexibility:

Why This Approach?

  • Pure CSS for the clock face and hands styling
  • Minimal JavaScript for time calculation and updates
  • Hardware Acceleration via CSS transforms ensures smooth animation
  • Small footprint with no external dependencies

Technical Breakdown

  1. Clock Face: Created with a circular div and positioned numbers
  2. Clock Hands: Absolutely positioned divs with transform-origin at the bottom
  3. Animation: JavaScript calculates the current time and updates hand positions every second
  4. Smooth Transitions: CSS transitions create natural hand movements

    <script>
    
        function setDate() {
            const now = new Date();
            
            // Analog clock
            const seconds = now.getSeconds();
            const secondsDegrees = ((seconds / 60) * 360) + 90;
            document.querySelector('.second-hand').style.transform = `rotate(${secondsDegrees}deg)`;
            
            const minutes = now.getMinutes();
            const minutesDegrees = ((minutes / 60) * 360) + ((seconds/60)*6) + 90;
            document.querySelector('.min-hand').style.transform = `rotate(${minutesDegrees}deg)`;
            
            const hours = now.getHours();
            const hoursDegrees = ((hours / 12) * 360) + ((minutes/60)*30) + 90;
            document.querySelector('.hour-hand').style.transform = `rotate(${hoursDegrees}deg)`;
            
            // Digital clock
            const digitalClock = document.getElementById('digitalClock');
            const timeString = now.toLocaleTimeString();
            digitalClock.textContent = timeString;
        }
        
        setInterval(setDate, 1000);
        setDate(); // initial call
    </script>

Best Practices

  • Use CSS transforms for smooth animations rather than JavaScript
  • Accessibility matters - Ensure proper contrast for clock hands
  • Performance optimized - Minimal JavaScript with hardware-accelerated CSS
  • Responsive design - Clock scales appropriately on different devices
  • Clean code - Well-structured and commented for easy maintenance
